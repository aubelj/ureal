------- utilisation de flo

installer nodeJs
installer l'extension flo disponible sur le store chrome
ouvrir outil de développement (F11)
dans l'onglet flo configurer avec l'adresse puis le port (8888)
dans la console lancer "node flo.js"
puis activate for this site dans l'onglet flo de l'outil de developpement
